CSS Shape 形狀的運用

基本形狀

  • 利用「區塊」與「邊框圓角」(border-radius) 製作出來的形狀。
  • 內容可以置入顏色、文字、圖片、影片等各種元素。
  • 設定最單純，使用上也最廣泛。

衍生形狀

  • 利用「邊框圓角」(border-radius) 的變化製作出來的形狀。
  • 內容可以置入其他元素，但會超出範圍。
  • 利用「邊框」(border) 的變化製作出來的形狀。
  • 內容沒有空間可以置入其他元素。
  • 設定最單純，使用上也最廣泛。

組合形狀

  • 利用「區塊」本身搭配前後「偽元素」製作出來的組合形狀。
  • 可以利用區塊內容空間來置入其他元素。
  • 設定稍微複雜，使用上需要多花點功夫與巧思。

利用 clip-path

  • clip-path 的概念類似 Photoshop 的遮色片，理論上可以將方形區塊裁切出任何幾何形狀。
  • 雖然區塊內容可置入其他元素，但須注意到會有被裁切的部分。
  • 設定上比較不直覺，搭配工具會比較快速。
  • 工具在這裡：https://bennettfeely.com/clippy/
